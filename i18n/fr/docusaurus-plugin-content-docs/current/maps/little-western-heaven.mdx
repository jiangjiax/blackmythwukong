# Carte de la Petite Occident dans Black Myth: Wukong

## Colline de Huangfeng dans le Voyage en Occident

Dans le roman original du Voyage en Occident, la Petite Occident est mentionnée pour la première fois au chapitre 65, intitulé "Les Démons Se Déguisent en Petit Monastère du Tonnerre ; Les Quatre Pèlerins Rencontrent Tous une Grande Calamité." C'est l'emplacement du Petit Monastère du Tonnerre.
