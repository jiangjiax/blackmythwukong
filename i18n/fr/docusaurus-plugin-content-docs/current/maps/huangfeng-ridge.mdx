# Carte de la Colline de Huangfeng dans Black Myth: Wukong

Dans Black Myth: Wukong, la Colline de Huangfeng est un lieu significatif inspiré du classique chinois Le Voyage en Occident.

## Colline de Huangfeng dans Le Voyage en Occident

Dans le texte original de Le Voyage en Occident, la Colline de Huangfeng est mentionnée pour la première fois au chapitre 20, intitulé "Colline de Huangfeng : Tang Seng en difficulté, Zhu Bajie fait la course en montagne." Elle est également appelée "la Colline de Huangfeng des Huit Cents Li."
