# Marée Écarlate dans Black Myth: Wukong

- **Santé:** 576
- **Type:** Transformation
- **Nom:** Marée Écarlate
- **Recharge:** Aucune
- **Coût en Mana:** Aucun

## Mouvements

**Coupe Lunaire (Attaque Légère)**

Une attaque légère, fait danser les flammes sur la lame comme une pleine lune, tranchant continuellement sur l'adversaire.

**Poursuite Lunaire (Attaque Lourde)**

Une attaque lourde, légère comme le feu, poursuivant la lune comme une météorite.

Après avoir esquivé dans différentes directions, suivre avec une attaque lourde pour évoluer en nouvelles variations.

**Briseuse de Lune (Compétence Passive)**

Après une esquive, la puissance d'attaque est considérablement augmentée pendant un court instant.

## Méthode d'Acquisition

Vaincre le boss Guang Zhi (Instructeur Loup-lame) dans le niveau de la Montagne du Vent Noir pour obtenir.

## Introduction Détaillée

Lors de l'utilisation, se transformer en démon loup, gagnant l'immunité au feu.

Les vagues de feu sont intenses et pressantes. Les attaques peuvent infliger la Peste de Feu.

Les mouvements infligent la Peste de Feu, qui peut brûler continuellement l'adversaire.

Pendant la transformation, entrer dans un état immunisé contre le feu.
