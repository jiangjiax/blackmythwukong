# 黒神話：悟空における玉塵

- **種類:** 変身
- **名前:** 玉塵
- **クールダウン:** --
- **マナコスト:** --

## 技

**クラッシュ（技）**

十分な勢いを蓄積して放出し、広範囲に大きなダメージを与える。

**リストア**

真の姿に戻り、近くの敵を地面からの衝撃波で負傷させる。

## 取得方法

シェフン崖ステージの石人を倒すことで獲得する。

## 詳細な紹介

使用すると、石の精霊に変身し、火、冷気、雷、毒に対する免疫を得る。

堅く重く、防御に重点を置いている。変身中に攻撃を受けると、勢いを継続的に蓄積できる。
