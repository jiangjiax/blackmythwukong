# 黒神話：悟空における山猿

- **Type:** 変身
- **Name:** 山猿
- **Cooldown:** --
- **Mana Cost:** --

## 技

**クローアタック（軽攻撃）**

軽攻撃技、4段階からなる：右クロー、左クローで前進、右クロー、左クローで地面を叩く。

**ハンマーストライク（重攻撃）**

重攻撃技、4段階からなる：両手で地面を4回叩き、強力な衝撃を発生させ、近くの敵を押しのける。

## 取得方法

## 詳細紹介

使用すると、運命に導かれた者は巨大な猿に変身し、背が高く、動きが速い。変身は近くの敵を全て押しのける。変身時間：30秒。
