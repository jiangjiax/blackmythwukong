# 黒神話：悟空における形集と気散

黒神話：悟空では、形集と気散の技は、白煙に変化して逃げ、再び攻撃する機会を求めることができるアクティブな呪文です。

- **マナコスト:** 30

## スキルポイントの配分

### 敏捷性

気変化中、移動速度がわずかに上昇します。

### 洞察力

各レベルで、気変化中に毎秒蓄積される貫通攻撃のクリティカルヒット率がわずかに上昇し、最大10スタックまで達成できます。

## 詳細な紹介

運命に導かれた個人は、妖怪や魔物の精髄を吸収し、自然に身振り技を理解する。

手の印と呪文を唱えた後、偽の形を残し、純粋なエネルギーに素早く分散し、軽快に動き、煙のように行き来する。

この技は、相手の鋭い刃を避けたり、厳しい陣形に誘い込むことができ、戦略的な動きが可能である。

再形成すると、突然のキックを放ったり、強力な一撃のために力を集めたりすることができる。相手が準備不足であれば、それによって油断させることができる。

気を散らして逃げることですでにコストを回収しており、反撃が成功すれば追加の利益をもたらす。
