# 黒神話：悟空における蓮華の雹

- **Type:** 変身
- **Name:** 蓮華の雹
- **Cooldown:** --
- **Mana Cost:** --

## 技

**切り裂く（軽攻撃）**

軽攻撃技、手に持つ刀を振り回し、相手を切り裂く。

**自己苦悩（重攻撃）**

重攻撃技、自分自身を傷つけ、体内の卵を刀に付着させる。充電後、放出し、最大4層まで。

後続の軽攻撃が相手に命中するたびに、卵の1層を消費する。卵は相手の傷口に付着し、成虫に孵化して休眠状態になり、一定期間後に死ぬ。

**勝利（スペル）**

相手がすでに毒虫に侵されている場合、それらに一斉爆発を命令する。

相手に即座に重傷を与えることができ、一部の虫の群れは回復して一定量の生命を回復する。

**退却（回復技）**

自分自身を完全に爆発させ、範囲ダメージを与える。

真の姿に回復する際、相手に未消費の毒虫を爆発させる勝利を実行する機会を得ることができる。

## 取得方法

紫雲山の洛花荘で関連ボスの道士ミを倒した後に獲得する。

## 詳細な紹介

使用すると、虫の妖怪に変身し、毒に対する免疫を獲得する。

悪意と凶悪さを持ち、他人にも自分にも危害を加える。攻撃は毒疫を引き起こすことができる。

技は毒疫を引き起こし、相手を継続的に毒にかけることができる。

変身中、毒に対する免疫状態に入る。
