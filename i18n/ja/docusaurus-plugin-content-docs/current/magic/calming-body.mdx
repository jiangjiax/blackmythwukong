# 黒神話：悟空における炎の輪

黒神話：悟空では、「炎の輪」は非凡な技であり、神秘的な陣を描いて小さな生き物から身を守り、飲酒の効果を高めるアクティブな呪文です。元の『西遊記』の第五十章に記述された「鎮体法」に触発されています。

- **種類:** 非凡な技
- **名前:** 炎の輪
- **クールダウン:** 90秒
- **マナコスト:** 60

## 詳細な紹介

運命に導かれた個人は、偶然、極寒の凍土の地でこの命を守る非凡な技を学びました。

杖を取り、地面に輪を描くだけで、寒さを防ぎ、休息を取り、敵への防御を準備できます。

輪の中に立つことで、杖の勢いが継続的に増加し、エネルギーの回復が加速します。この間に温かいお酒を飲むと、さらに効果が大きくなります。

炎の輪は比較的長い持続時間があり、短時間戦闘に出ても、いつでも輪に戻って別の機会を探すことができます。

時間制限に達すると、陣は自然に消えます。
