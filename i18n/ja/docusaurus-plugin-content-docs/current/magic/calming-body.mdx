# 黒神話：悟空における鎮静の呪文

黒神話：悟空では、「鎮静の呪文」は特別な技法であり、神秘的な陣を描くことで小さな生き物を寄せ付けず、飲酒の効果を高めるアクティブな呪文です。元の「西遊記」の第五十章で述べられている「鎮体法」に触発されています。

- **種類:** 特別な技法
- **名前:** 鎮静の呪文
- **クールダウン:** 90秒
- **マナコスト:** 60

## 詳細な紹介

運命に導かれた個人は、偶然、極寒の地域でこの命を守る特別な技法を学びました。

杖を取り、地面に輪を描くだけで、寒さを寄せ付けずに休息し、敵への防御を準備できます。

輪の中に立つことで、杖の勢いが継続的に増加し、エネルギーの回復が加速されます。暖かいお酒をこの間に飲むと、さらに効果が大きくなります。

炎の輪は比較的長い持続時間があり、短時間戦闘に出ても、いつでも輪に戻って別の機会を探すことができます。

時間制限に達すると、陣は自然に消え去ります。
