# 黒神話：悟空における鉄頭と鋼の腕

黒神話：悟空では、鉄頭と鋼の腕の技は、身体を瞬時に暗い鉄のように硬くし、物理攻撃に対する抵抗力を提供するアクティブスペルです。

- **クールダウン:** 10秒
- **マナコスト:** 30

## 詳細な紹介

運命に導かれた個人は、非凡な危険や冒険に遭遇し、深い理解を通じて防御技術を習得します。

肩をすくめ、背中をまっすぐにすることで、金と石の塊に変化します。もし相手がこの硬化した形態を攻撃すると、その衝撃で手首が痛むことがあります。

相手が衝撃で跳ね返され、油断した場合、追撃の絶好のチャンスとなります。

この技は見た目が鈍く見えますが、効果的にするためには正確なタイミングが必要です。早すぎたり遅すぎたりすると、無駄になる可能性があります。

一部の相手は頑丈で勇敢であり、彼らの強力な一撃は鉄頭と鋼の腕をも突破することができます。
