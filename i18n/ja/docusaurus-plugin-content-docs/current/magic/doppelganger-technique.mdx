# 黒神話：悟空における多重影分身

黒神話：悟空では、「多重影分身」は自分の体から抜き取った毛を使って複数の分身を召喚し、戦闘を支援するアクティブスペルです。

- **種類:** 毛
- **名前:** 多重影分身
- **クールダウン:** 60秒
- **マナコスト:** 130

## 詳細な紹介

このスペルは、選ばれし者が自分の体から抜き取った毛を使用して発動します。毛を一把抜き、口の中で噛み砕き、空中に吐き出すことで、多くの毛猿が召喚されます。

これらの毛猿は俊敏で機敏であり、選ばれし者自身のように戦いや逃げることができます。主人が脅かされたときには相手と戦い、息抜きや戦略的撤退の機会を提供します。しかし、これらの毛猿の命は脆いものであり、強力で力強い敵に直面すると、すぐに滅ぼされるリスクがあります。
