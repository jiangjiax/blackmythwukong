# 黒神話：悟空における拘束

黒神話：悟空では、拘束は特別な技であり、標的となるモンスターを動けなくすることができるアクティブな呪文です。

- **タイプ:** 神秘の技
- **名前:** 拘束呪文
- **クールダウン:** 30秒
- **マナコスト:** 60

## スキルポイントの割り当て

### 脆弱性

スタシスの影響を受けた標的にヒットするたびに、与えるダメージがわずかに増加します。

### 凝固

金剛棒で相手にヒットすると、次のスタシス効果の持続時間をわずかに延長できます。スタシスごとに最大5層のスタシス延長を積み重ねることができます。

### 反射

相手が攻撃する瞬間に、それをスタンさせることができます！これにより、スタシスの持続時間が延長され、相手が受けるダメージが大幅に増加します。

### 不壊

スタシスの影響を受けた相手のスタシスが破られると、少量のマナを回復できます。

## 詳細な紹介

運命に選ばれた者の手のひらには、金色の「定」の文字が刻まれており、拘束の技を操る特別な能力を与えています。

指先をちょっと指して呪文を唱えるだけで、相手を動けなくすることができます。

しかし、偉大な神通力を持つ強敵は、この拘束からより迅速に逃れることができるため、注意が必要です。

戦いが激しく、相手の攻撃が絶え間ない場合、長時間拘束することは難しく、あまり有利ではありません。

この技を使うのに適した瞬間は、相手が静止している、倒れている、または弱体化して押し戻されている時かもしれません。
