# 紅潮 黒神話：悟空

- **体力:** 576
- **タイプ:** 変身
- **名前:** 紅潮
- **クールダウン:** なし
- **マナコスト:** なし

## 技

**月光斬り（軽攻撃）**

軽攻撃技、刃に炎を舞わせ、満月のように相手に連続斬りかかる。

**月追い（重攻撃）**

重攻撃技、炎のように軽く、流星のように月を追う。

異なる方向に回避した後、重攻撃で新しいバリエーションに進化させる。

**月破り（パッシブ能力）**

回避後、短時間だけ攻撃力が大幅に上昇する。

## 取得方法

黒風山のステージでボスの広之（刀狼教官）を倒して取得する。

## 詳細な紹介

使用すると、狼の妖怪に変身し、火に対する免疫を獲得する。

炎の波は激しく迫る。攻撃は火疫を引き起こすことができる。

技は火疫を引き起こし、相手を継続的に燃やすことができる。

変身中、火に対する免疫状態に入る。
