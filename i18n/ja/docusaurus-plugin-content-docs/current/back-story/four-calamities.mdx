# 四大災難

四大災難は、宇宙における四大主要な災害、すなわち雷撃、火災、極寒、毒によるものを指します。これらの災害が一定の程度に達すると、それに応じたダメージを与えます。薬草を服用することで、影響を軽減することができます。

## 歴史的背景

四大災難の概念は、中国の小説「西遊記」の第二章で言及されている元の三大災難から適応された可能性があります。三大災難は、道教の修行者が修業中に直面しなければならない三つの災害、すなわち雷撃、火災、風を指します。これらの災害は単なる物理的な挑戦ではなく、修行者の決意と純粋さを試す精神的な試練でもあります。この文脈での第四の災害、毒の追加は、物語の中のキャラクターが直面する挑戦と危険の範囲を拡大するための適応かもしれません。
