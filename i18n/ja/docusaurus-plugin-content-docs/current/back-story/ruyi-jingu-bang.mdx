# 如意金箍棒

如意金箍棒は、明代の小説「西遊記」に登場する孫悟空が使用する武器であり、神々と妖怪たちの物語を描いた中国の古典的なファンタジーです。

## 紹介

小説における孫悟空の武器の正式名称は「如意金箍棒」で、他にも金箍如意棒、金箍鉄棒、如意棒、金箍棒、凌陽棒、鉄棒などの別名があり、最もよく使われる略称は鉄棒です。「西遊記」によると、如意金箍棒は「両端に金の帯がつけられ、中間は暗い鉄でできている。帯の近くには『如意金箍棒』という文字が刻まれ、重さは13,500斤である」と描写されています。東海竜宮の竜女と竜姫はこれを「神珍鉄」と呼んでいます。王勃の「広州宝庄寺塔碑銘」には「誠心が神秘的に動き、神珍が現れる」とあります。孫悟空が神珍を得た後、これは彼の思いに応じて大きさや長さを変えることができ、本当に「如意」（思いのまま）という名前にふさわしいものになりました。

## 起源

「金環杖説」。「大唐三蔵法師詩話」では、猿行者は武器を持たず、大梵天王から授かった金環杖で妖怪を捕らえています。

「金剛杖と金環杖の混合説」。日本の学者たちは、密教の金剛杖のイメージが「西遊記」の孫悟空のイメージに影響を与え、金箍棒の創造につながったと考えています。

「金棒と鉄棒説」。元代の戯曲「西遊記」では、孫悟空は形を変えることができる金棒を使っています。「西遊記平話」では、孫悟空は鉄棒を使っており、これは「西遊記」に直接影響を与えました。

「金骨棒説」。周汝昌氏は、「金箍棒」は実は「金骨棒」であり、発音の間違いによって「金箍」と間違えられたと考えています。

「如意珠説」。「如意珠」は仏教の宝の一種で、「摩尼珠」とも呼ばれます。この珠を持つと、世の中のすべてのものを見ることができ、思いのままに物を変えることができ、すべての災難と苦しみを消し去ることができるとされています。「大智度論」によると、この珠は竜王の頭脳から生まれます。「大方便仏報恩経」には、竜王は如意珠を左の耳に入れていると書かれています。
