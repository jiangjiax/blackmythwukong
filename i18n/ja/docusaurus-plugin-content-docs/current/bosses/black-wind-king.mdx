# 黒神話：悟空における黒風王

黒風王は、黒風山の黒風洞に位置する強力な妖怪王です。彼は黒風山の支配者であると主張し、強力な力と激しく巧みな戦闘スタイルを見せつけます。かつては普通の妖怪だった黒風王は、欲望と権力欲に堕ち、最終的に黒風山の妖怪王へと変貌しました。彼の物語は欲望と堕落のテーマを反映し、権力追求における自己の喪失を象徴しています。

## 戦略

黒風王の戦闘スタイルは、高い爆発ダメージと俊敏な攻撃が特徴です。プレイヤーは戦闘中常に警戒し、環境を利用して回避と反撃を行う必要があります。彼の攻撃パターンを観察することで、プレイヤーは反撃の機会を見つけながら、彼が召喚する炎と嵐から身を守ることができます。

## 技

### 叩きつけ攻撃

黒風王は地面を強く叩きつけ、範囲ダメージを与えます。プレイヤーは迅速に回避して被弾を避けるべきです。

<img src="https://image.informedainews.com/bwk1.gif" alt="黒神話：悟空"/>

### 炎の拳

彼は強力な左パンチを繰り出し、次に右の爪で前方に突進して敵を捕らえて消費しようとします。この技は回復時間が長いため、プレイヤーは反撃してダメージを与えることができます。

<img src="https://image.informedainews.com/bwk2.gif" alt="黒神話：悟空"/>

### 地面炎噴出

黒風王は地面を叩き、その下から炎が噴出します。プレイヤーは迅速に回避して焼かれるのを避ける必要があります。

<img src="https://image.informedainews.com/bwk3.gif" alt="黒神話：悟空"/>

### 黒風突撃

彼は暗い風に変化し、急速に移動してプレイヤーに突撃し、大きなダメージを与えます。プレイヤーは風の動きに注意し、適切に回避する必要があります。

<img src="https://image.informedainews.com/bwk4.gif" alt="黒神話：悟空"/>
