# 黒神話：悟空の金池長老

金池長老は観音院の住職です。玄奘一行が通りかかった際、玄奘の錦の袈裟を見て欲深くなり、最終的に寺院が焼け落ちる中で命を絶ち、妖怪王ボスに変化しました。彼の身分と経験は、貪欲と罪のテーマを反映しており、彼の人生での誤った選択が、現在の妖怪王ボスという地位に至ったことを示唆しています。

## 攻略

金池長老の戦闘スタイルは、高いダメージと範囲攻撃が特徴です。戦闘中、プレイヤーは闘技場の修行僧を倒すことで体力を回復でき、難易度を下げることができます。全体的に、金池長老の戦闘メカニクスは幽霊と似ていますが、高いダメージ出力と召喚能力のため、プレイヤーは戦闘戦略により注意を払う必要があります。

## 技

### 両掌連打

金池長老は静止したまま両掌で連続攻撃を行い、大きなダメージを与えます。攻撃を避けるために距離を取ることをお勧めします。

### 跳躍、叩きつけ、昇雲掌

金池長老がプレイヤーから離れていると、跳躍して地面を叩きつけ、衝撃波と連続攻撃を引き起こします。プレイヤーは最初の叩きつけを避けて大きなダメージを回避する必要があります。

### 突進、叩きつけ、螺旋出力

予測しやすく避けやすい遅い飛び蹴り攻撃です。

### 爆発する修行僧を召喚

金池長老は修行僧を召喚し、プレイヤーを追いかけて爆発させ、ダメージを与えます。プレイヤーはこれらの修行僧を優先的に排除し、金池長老が体力を回復するのを防ぐべきです。
