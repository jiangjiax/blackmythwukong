# 黒神話：悟空における百足怪

黒神話：悟空では、百足怪は試練イベントの千紅洞地区のボスであり、「指導的ボス」としての役割を果たしています。

## 攻略

百足怪が召喚する虫の群れは動きが遅いです。洞窟を素早く走り回ることで、それらを引き離すことができるかもしれません。

攻撃されるときは回避し、転がるときは追いかけ、咆哮するときは逃げる。機会を見つけたら？コンボで懲らしめるだけです！

## 技

### 固定開始

#### 跳躍

最初に、百足怪は遠くから跳びかかります。少し遅めに回避することで避けることができます。プレイヤーがキャラクターを前進させて転がすと、百足怪の後ろに着くでしょう；横に転がると、通常は横や百足怪の前に着くでしょう。

跳躍を回避した後、軽い攻撃で反撃することができますが、通常は完全な軽い攻撃コンボを完了するのに十分な時間はありません。その後、百足怪はプレイヤーの位置に基づいて後続の攻撃を開始します。

（跳躍は専用の開始技ではないことに注意してください；通常の戦闘中、プレイヤーが百足怪から遠く離れている場合、跳躍を引き起こすこともあります。）

<img src="/img/bosses/feipu.gif" alt="Black Myth: Wukong"/>

#### 三連打

プレイヤーが跳躍を回避し、百足怪の頭の前に留まると、頭で二回打ちつけ、その後に一回背中で打ちつけますが、頭の打ちつけよりも遅いです。

<img src="/img/bosses/sandusnza.gif" alt="Black Myth: Wukong"/>

#### 背中打ちつけ

プレイヤーが跳躍を回避し、百足怪の後ろに留まると、直接背中で打ちつけを開始します。

二回の頭打ちつけ後の背中打ちつけと同じ方法で対処します。

<img src="/img/bosses/beiza.gif" alt="Black Myth: Wukong"/>

### 通常戦闘

#### 遠距離毒液噴射

より遠い距離で、百足怪は前方に毒液を噴射します。プレイヤーは少し動かすことでほとんどのダメージを避けることができます。

<img src="/img/bosses/yjlpd.gif" alt="Black Myth: Wukong"/>

#### 第一段階虫転がし

距離がある場合、百足怪は丸くなってプレイヤーに向かって3-5回素早く転がります。

プレイヤーは「鉄の意志」を使って転がしの開始段階で止めることを選ぶことができます。転がしが十分な加速を得ている場合、「鉄の意志」で止めることはできませんが、衝突後も転がり続けます。

百足怪がすでに加速している場合、三回目の転がしがプレイヤーにとっての懸念点です。第一段階（群れを召喚する前）では、転がしは三つのセグメントに固定されています。三番目のセグメントがプレイヤーを外れると、壁に当たって止まり、プレイヤーに攻撃する時間を与えます；三番目のセグメントがプレイヤーに当たると、百足怪はドリフトしてアリーナで止まり、すぐに次の動きを続け、プレイヤーに攻撃する時間を残しません。

<img src="/img/bosses/chonche.gif" alt="Black Myth: Wukong"/>

#### 中距離突進

中距離（約転がしの長さ）で、百足怪は体を弓なりにして前方に二回突進し、二回目の突進は強い方向修正を持ち、壁に当たるまで止まりません。

「豆」が二つ以上ある場合、突進攻撃を避けるために素早く杖を立て、その後に三つの豆で突進して反撃することを検討できます。

<img src="/img/bosses/zjltj.gif" alt="Black Myth: Wukong"/>

#### 近距離毒液噴射

より近い距離で、百足怪は下方向に毒液を噴射します。プレイヤーは動かしてダメージを避けることができます。

<img src="/img/bosses/jjlpd.gif" alt="Black Myth: Wukong"/>

### 第二段階

#### 現地詠唱

百足怪は長い咆哮を放ち、数十匹の小さな虫を召喚し、球形に丸まってプレイヤーに向かって転がります。プレイヤーは杖の充電方法や杖の回転を使ってブロックすることができます。
