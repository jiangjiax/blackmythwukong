# 黒神話：悟空におけるリングスー

リングスーは、黒風山の観音寺に位置する妖怪王ボスです。彼は巨大な灰色の狼の精霊であり、黒風王の仲間です。

キャラクターのデザインと能力は、伝統的な中国神話からインスピレーションを受けており、リングスーは強力な存在感と黒風王との関連で知られる狼の精霊「リングスー」に関連付けられています。

## 攻略法

リングスーの攻撃は強力ですがシンプルで、最大4回のコンボ攻撃を行います（2020年のゲームプレイデモンストレーションに基づく）。彼の攻撃は起動時間と回復時間が長く、プレイヤーが十分な時間をかけて反応することができます。彼は火属性に非常に弱く、ナイフの狼のインストラクター「広志」は、各攻撃がボスを中断させてダメージを与えるため、優れた変身の選択肢となります。

灰色の狼の精霊として、リングスーは厚い毛皮を持ち、火に対して脆弱です。

## 技

### 引き裂く爪

これは開始の技で、リングスーが前方に2歩進み、その後ジャンプして素早く腕を広げて攻撃します。

<img src="https://image.informedainews.com/siliezhua.gif" alt="黒神話：悟空"/>

### 跳躍掃引

高くジャンプした後、リングスーは右前肢で掃引し、着地時に左前肢で広範囲に掃引します。

<img src="https://image.informedainews.com/fpjhs.gif" alt="黒神話：悟空"/>

### 狼の遠吠え

リングスーは数歩下がり、遠吠えをして広範囲にダメージを与えます。

<img src="https://image.informedainews.com/lh.gif" alt="黒神話：悟空"/>

### 噛みつき

プレイヤーがリングスーに近づくと、彼は前方に飛びかかって噛みつきます。この攻撃は単一のアクションで構成され、下に転がることで簡単に回避できます。
