# 黒神話：悟空におけるブレードウルフインストラクター

ブレードウルフインストラクターは、ゲーム「黒神話：悟空」に登場する黒風山のリーダーで、本名は「広智」です。

広智は、原作「西遊記」の第十六章「観音院の僧が宝を企む、黒風山の怪物が袈裟を盗む」に登場します。彼は観音院の金池長老の弟子です。唐僧とその弟子孫悟空が院に到着すると、金池長老は彼らを温かく迎え入れます。悟空は自分の袈裟を長老に見せびらかし、長老はそれを欲しがり、部屋で鑑賞するために借ります。長老は袈裟を手に入れたいと考え、広智の提案を受け入れて唐僧を刀で殺そうとしますが、失敗します。その後、別の弟子である広謀のアドバイスを聞き、僧たちに火を放つよう命じ、唐僧とその弟子を焼き殺そうとします。悟空は南天門に行き、広目天の火傷防止カバーを借りて師匠を守り、神風を吹かせて観音院を瓦礫の山にします。恥ずかしさと恐怖に襲われた長老は顔をあわせられず、壁に頭を打ちつけて死にます。

## 攻略

スキルを使わせないように圧倒する。

## 技

### ダッシュとダブルスラッシュ

ブレードウルフインストラクターは静止して力を溜め、両刃の剣を振るう準備をします。そして、プレイヤーの近くまで前進し、振るい、続けて2回の連続した下向きの攻撃を行います。各攻撃は短い追跡動作を伴います。この技の最初のダッシュは長い予備動作があるため、避けやすいですが、2回目の攻撃は速さと遅さのブレードの疑いがあるため、避ける際と反撃する際に注意が必要です。3回目の攻撃の2回目の回転では、手に持つブレードを回転させ、体の周りに360度の攻撃判定を作り出します。
