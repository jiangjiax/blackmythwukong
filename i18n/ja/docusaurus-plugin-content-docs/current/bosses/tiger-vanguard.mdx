# 黒神話：悟空におけるトラ先鋒

数百年前、前代のトラ先鋒は玄奘を捕らえようとして命を落とした。彼の残した二人の息子は、自分たちで生き抜くしかなかった。兄であるトラ兄は、残された小さな妖怪たちを率いて荒れ果てた山を進み、生き延びるために苦労した。幸い、彼らの父は彼らにいくつかの本当の技術を教えてくれたので、彼らは耐えることができた。

砂嵐が収まると、母石と石の妖精たちは山を以前の栄光に戻した。トラ兄は父のやり方に従い、山の中の怪物たちを管理し、それぞれが役割を持ち、生き物を無差別に傷つけないようにした。その後、いくつかの人間が山に村を建て、トラ兄は「悪しき侵入者を食べ、善き旅人を導く」という習慣に従い、「トラ神」という称号を得た。

村人たちは「トラ神」を祀るために蹲虎寺を建てたが、これによりトラ兄の弟であるトラ弟はますます不満を感じ、兄がただ早く生まれただけですべての名声を独り占めしていると感じた。

さらに百年後、大聖猿が突然鼠の妖怪たちと共に黄風峠に戻ってきた。トラ兄は不機嫌になり、その黄毛の鼠の精がまさに彼らの父が命を落とした原因だと言った。彼はトラ弟に鼠の妖怪たちを追い払うように仕向け、トラ弟はそれが理にかなっていると思い、同意した。

しかし、戦いの中で、大聖猿は仏の頭に似た武器を振るい、風から刃を召喚し、トラ兄を一撃で重傷させた。兄の命を救うために、トラ弟は武器を捨てて跪くしかなかった。大聖猿は恨みを持たず、兄弟たちの歴史を知り、トラ弟を自分の指揮下の先鋒に任命した。

トラ弟は回復したが、兄が先鋒になったことを知ると、従わないことを拒否した。二人のトラは大喧嘩をして、トラ弟は怒りに駆られて兄と何も関わらないことを拒否した。

トラ先鋒にとって、その黄毛の鼬鼠は彼に家族を二度失わせたのだから、本当に憎らしい。しかし、毎日もっと鼠を食べて怒りを発散する以外に彼にできることは何だろうか？

### 三連フックパンチからスラムへ

トラ先鋒は構えを取り、右腕を軽く引き戻し、前方に体を傾けて回転キックを放ち、距離を詰め、直後に連続して二つの前方パンチ（右、次に左）を放つ。短い溜めの後、空中に飛び上がり、叩きつける。

トラ先鋒の回転キックはかなりの動きがあり、プレイヤーに素早く圧力をかけることができる。回避のタイミングと方向に注意。二つのパンチコンボを完璧に回避した後、スラムの回復ウィンドウを待って罰する。
