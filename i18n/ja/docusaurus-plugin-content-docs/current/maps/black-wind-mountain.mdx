# 黒神話：悟空における黒風山マップ

**黒神話：悟空**の黒風山マップページへようこそ！この険しい地域は、強敵、豊富なルアー、そして様々なコレクティブルの巣窟であり、プレイヤーにその奥深くを探検し、秘密を解き明かすことを促します。

## 黒風山のエリア

黒風山はいくつかの異なるエリアに分かれており、それぞれ独自の課題と敵が待ち受けています：

- 狼の森
- 竹林
- 古代の観音寺
- 黒風洞

## 黒風山のボス

黒風山を進む中で、プレイヤーは様々なボスに遭遇し、それぞれ独自の挑戦を提供します：

- [刀狼教官](/docs/blackwindking)
- [リンシューズ](/docs/lingxuzi)
- 白衣の貴族
- [黒風王](/docs/blackwindking)
- [金智長老](/docs/elderjinchi)

## 黒風山のコレクティブル

プレイヤーは黒風山で様々なコレクティブルを集めることができます：

- 霊芝（霊芝茸）
- ヒレシメ（獅子髪茸）
- アコニチウム（僧帽花）
- ハス（蓮）
- ヒレシメ（獅子髪茸）
- ウルフイポリア（タカハコ）
