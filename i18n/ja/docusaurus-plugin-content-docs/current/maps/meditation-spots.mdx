# 黒神話：悟空の瞑想スポット

ゲームには、プレイヤーが休憩し、初めてそこで瞑想することでスパークを獲得できる一連の瞑想スポットがあります。すべての瞑想スポットの場所は以下の通りです：

## 黒風山

- 森のベンチ、狼の森 - 狼の森を見下ろす荒れ果てた木造建物の中

- 洞窟、竹林 - 竹林の奥の丘の洞窟内

- 崖、黒風洞 - 黒風洞の谷を見下ろす木製のプラットフォーム上

## 黄風峠

- 祭壇、砂門村 - 砂門村の仏教の祭壇の前

- 谷、壊れた石のテラス - 壊れた石のテラスの前の谷

- 像、うずくまる虎寺 - うずくまる虎寺の中の像の下

- 崖、枕石のテラス - 枕石のテラスの枯れた松の木の下

- 洞窟、黄風の陣 - 黄風の陣の仏教の洞窟の前

- 岩、砂門の境界 - 砂門の境界の岩の上

## 新西域

- 日陰、ミラーミア - ミラーミアを見下ろす大きな木の下

- 底、塔の世界 - キャプテン・ワイズボイスを倒した後の塔の世界の底

- 像、教義の回廊 - 教義の回廊の仏像の前

- 道、瞑想の崖 - 瞑想の崖の金色の壁画に続く道

- 広間、新しい雷音寺 - 新しい雷音寺の寺の広間内

## 綿糸峠

- 崖、壊れた翡翠の池 - 壊れた翡翠の池を見下ろす崖

- 日陰、中層 - 中層の日陰の木の下

- 奥深く、最下層 - 最下層の洞窟の奥深く

- 崖、虎蛇虫の森 - 虎蛇虫の森の崖

- 松、黄華の亭 - 黄華の亭の松と日傘の木の間

- 岩、紫雲山 - 紫雲山の岩の上

## 炎の山

- 彫刻、四季のキャンプ - 四季のキャンプの彫刻の上

- 仏、エメラルドの広間 - エメラルドの広間の座った仏の前

- 岩、灰の峠III - 灰の峠IIIの岩の上

- 洞窟、精錬の穴 - 精錬の穴の石の屏風の前

24の瞑想スポットをすべて見つけると、景色探求者のアチーブメント/トロフィーが解除されます。各スポットで初めて瞑想することでスパークが与えられ、スキルのアンロックに使用できます。
