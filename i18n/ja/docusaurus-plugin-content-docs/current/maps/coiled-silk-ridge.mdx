# 黒神話：悟空における螺旋の絹の尾根の地図

## 西遊記における螺旋の絹の尾根

西遊記の原作では、螺旋の絹の尾根は第72章「絹の洞：七情が元の性質を惑わす；洗い垢の泉：猪八戒が自制を失う」で初めて言及されています。

地元の土地神が孫悟空にその地域を紹介し、螺旋の絹の尾根の下には絹の洞という洞窟があり、そこには七人の女性の魔物が住んでいると述べています。絹の洞の南には、洗い垢の泉という泉があり、これは本来七仙女の入浴場であった天然の温泉ですが、現在は魔物に占領されています。
