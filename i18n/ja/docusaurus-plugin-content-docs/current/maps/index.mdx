## 黒神話：悟空 マップインデックス

**黒神話：悟空**のマップインデックスページへようこそ！ここでは、ゲーム内の様々なマップ、敵の紹介、隠されたエリア、戦略のヒントに関する詳細な情報を見つけることができます。黒神話：悟空には合計8つのマップがあり、それらは以下の通りです：

- [黒風山](/docs/blackwindmountain)
- 花果山
- [黄風嶺](/docs/huangfeng-ridge)
- [小西天](/docs/little-western-heaven)
- 紫雲山
- [繰糸嶺](/docs/coiled-silk-ridge)
- 火焰山

これらの多様な場所を探索し、その秘密を解き明かし、**黒神話：悟空**の神秘的な世界で冒険に出る際に強敵との刺激的な戦いに備えましょう！

## [瞑想スポット](/docs/meditationspots) 黒神話：悟空

多様なマップに加えて、**黒神話：悟空**には複数の[瞑想スポット](/docs/meditationspots)があり、プレイヤーはそこで休息し、力を回復することができます。これらの[瞑想スポット](/docs/meditationspots)はしばしば隠れた角や寺院に位置しており、プレイヤーに息抜きの機会を提供します。ここでは、体力を回復し、装備を整理し、次の挑戦に備えることができます。
