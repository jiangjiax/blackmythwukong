# 黒神話：悟空における黄风岭マップ

黒神話：悟空では、黄风岭は古典的な中国の小説『西遊記』にインスパイアされた重要な場所です。

## 西遊記における黄风岭

『西遊記』の原文では、黄风岭は第20章「黄风岭：唐僧の危機、猪八戒山を競う」で初めて言及されており、「八百里黄风岭」とも呼ばれています。
