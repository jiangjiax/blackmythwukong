# الإحصائيات

تشمل السمات الحياة، والمانا، والصحة، إلى جانب السمات الأساسية ومقاومات الكوارث الأربع.

السمات الأساسية: الهجوم، الدفاع، نسبة الضربات الحرجة، مضاعفة الضربات الحرجة، تعزيز الضرر، وتقليل الضرر.

مقاومات الكوارث الأربع: مقاومة البرد، مقاومة النار، مقاومة السموم، ومقاومة البرق.
