# 龙

龙是中国官方、宗教和民间信仰中共同崇拜的传说生物，在文学创作中享有极高的知名度。

## 黑神话悟空中的龙

根据游戏的宣传视频，我们了解到亢金龙和泾河龙王的两个儿子赤髯龙和小鲤龙。

## 西游记中的龙

在《西游记》中，最早出现的龙是四海龙王，孙悟空为了获取武器和盔甲而拜访他们。在得到金箍棒后，他又进一步请求一套好盔甲，这导致了其他三位海龙王的到来。接下来是泾河龙王，他在梦中向唐太宗求助。

随后，故事介绍了白龙，即玉龙三太子，在鹰愁涧被观音菩萨所救，以及托龙，他捉住了唐三藏，从而引入了泾河龙王的九个儿子。

在小说中，龙通常作为掌管雨水和水体的神祇，每条河流和井都有自己的龙王。此外，他们还被描绘为玉帝的坐骑或作为其他神祇的守护者和使者。

## 现实世界中的龙

考古发现表明，龙图腾，包括鱼龙、蛇龙、猪龙和鳄龙，早在7000年前就因中国的原始自然崇拜而出现。现代龙的特征是鹿角、驼头、兔眼、蛇颈、蜃腹、鱼鳞、鹰爪、虎掌和牛耳。这种独特的形象与神祇的使者或神圣坐骑等角色相关联。

需要注意的是，龙控制风雨的能力与佛教的中国化密切相关。然而，龙作为雷神的崇拜在6000年前的文献中就有记载，包括《山海经》、《五帝本纪》和《淮南子》。

在佛教经典中，如晋代翻译的《妙法莲华经》中，有八位龙王，包括掌管降雨的海龙王娑伽罗龙王。然而，应该注意的是，佛教文本中的龙通常指的是那伽，这些蛇形生物在印度神话中早于佛教存在。它们被纳入佛教，后来被翻译成中文为龙或蛇。随着佛教在中国化的过程中，寺庙中这些神祇的形象开始与印度起源有所不同，并在流行文化中融入为中国特有的形式。
