# 黑神话悟空中的盘丝岭地图

## 西游记中的盘丝岭

在《西游记》原著中，盘丝岭首次出现在第七十二回，标题为“丝洞七情迷本性，濯垢泉八戒忘形”。

当地土地向孙悟空介绍该地区，提到盘丝岭下有一个名为丝洞的洞穴，居住着七名女妖。在丝洞的南边，有一处名为濯垢泉的温泉，这是一处天然温泉，原本是七仙女的沐浴池，后被妖怪占据。
