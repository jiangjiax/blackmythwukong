# 黑神话悟空中的冥想点

游戏中设有一系列冥想点，玩家首次在此冥想可获得一颗火花。以下是所有冥想点的位置：

## 黑风山

- 狼林树屋 - 俯瞰狼林的破旧木结构内部

- 竹林后山洞穴 - 竹林后山洞穴内

- 黑风洞悬崖 - 黑风洞中俯瞰深渊的木制平台

## 黄风岭

- 沙门村祭坛 - 沙门村佛坛前

- 碎石台地谷 - 碎石台地前的谷地

- 卧虎寺雕像 - 卧虎寺内雕像下

- 枕石台地悬崖 - 枕石台地枯松树下

- 黄风阵佛窟 - 黄风阵佛窟前

- 沙门边界岩石 - 沙门边界岩石上

## 小西天

- 镜海树荫 - 俯瞰镜海的大树下

- 塔界底层 - 击败智音队长后的塔界底层

- 戒律廊佛像 - 戒律廊佛像前

- 正念崖小径 - 通往正念崖金色壁画的小径上

- 新雷音寺大殿 - 新雷音寺大殿内

## 盘丝岭

- 碎玉池悬崖 - 俯瞰碎玉池的悬崖上

- 中层树荫 - 中层树荫下

- 底层深处 - 底层洞穴深处

- 虎蛇虫林悬崖 - 虎蛇虫林悬崖上

- 黄花亭松树 - 黄花亭松树与伞树之间

- 紫云山岩 - 紫云山岩上

## 火焰山

- 四季营地浮雕 - 四季营地浮雕上

- 翡翠殿坐佛 - 翡翠殿坐佛前

- 灰烬通道III岩 - 灰烬通道III岩上

- 炼化坑石屏 - 炼化坑石屏前

找到所有24个冥想点可解锁“风景探索者”成就/奖杯。首次在每个冥想点冥想可获得一颗火花，用于解锁技能。
