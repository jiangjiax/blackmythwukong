# 赤潮在黑神话悟空

- **生命值：** 576
- **类型：** 变身
- **名称：** 赤潮
- **冷却时间：** 无
- **法力消耗：** 无

## 招式

**月斩（轻攻击）**

轻攻击招式，刀尖上的火焰如满月般舞动，连续斩击对手。

**逐月（重攻击）**

重攻击招式，轻盈如火，追逐月亮的流星。

在不同方向闪避后，接重攻击可衍生出新的变化。

**月碎（被动能力）**

闪避后，短时间内攻击力大幅提升。

## 获取方式

击败黑风山关卡的Boss广智（刀狼教头）获得。

## 详细介绍

使用后，变身为狼妖，获得火焰免疫。

火浪汹涌，压迫感十足。攻击可施加火瘟。

招式可施加火瘟，持续灼烧对手。

变身期间，进入火焰免疫状态。
