# 黑神话悟空中的虎先锋

数百年前，前代虎先锋在捉拿玄奘时丧命，留下两个儿子相依为命。长兄虎哥带领残余的小妖们穿越荒废的山岭，艰难求生。幸好父亲曾传授他们一些真本事，让他们得以坚持下来。

当沙暴平息，母石与石精们将山岭重建至昔日的辉煌。虎哥效仿父亲，管理山中的妖怪群体，确保每个妖怪都有其职责，不随意伤害生灵。后来，一些凡人在山中建村，虎哥遵循“吞噬恶行者，引导善行人”的习俗，赢得了“虎神”的称号。

村民们建造了卧虎寺来供奉“虎神”，这让虎哥的弟弟虎弟越来越不满，觉得长兄只是因为早出生几年就独占了所有威望。

又过百年，齐天大圣突然带着一群鼠妖回到黄风岭。虎哥不满，称正是因为这个黄毛鼠精，他们的父亲才丧命。他煽动虎弟赶走鼠妖，虎弟觉得有理，便同意了。

然而，在战斗中，齐天大圣挥舞着形似佛头的武器，召唤风刃，一击重伤虎弟。为了救兄长的性命，虎弟不得不放下武器，跪地投降。齐天大圣不记仇，得知兄弟俩的过往后，任命虎弟为麾下先锋。

虎弟伤愈，但得知兄长成为先锋后，拒绝服从。两虎大吵一架，虎弟愤怒离去，誓不再与兄长有任何瓜葛。

对于虎先锋来说，那个黄毛鼬鼠精让他两次失去了家人，实在可恨。但除了每天多吃几只老鼠泄愤，他还能做什么呢？

## 招式

### 冲锋猛击

虎先锋先肩撞一段距离，然后跳起猛击地面。此招相对容易完美闪避，猛击后有暴露关节动画。

### 三连下勾拳接猛击

虎先锋摆出架势，轻拉右臂后，身体前倾，施展旋转踢快速接近，紧接着连续两记前冲拳（右、左），短暂蓄力后跳起猛击地面。

虎先锋的旋转踢移动距离较大，能快速压迫玩家。注意闪避的时机和方向。完美躲过两记拳组合后，等待猛击的恢复窗口进行反击。
