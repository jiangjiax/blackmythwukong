# 검은 신화: 오공의 옥 가루

- **타입:** 변신
- **이름:** 옥 가루
- **쿨다운:** --
- **마나 비용:** --

## 기술

**분쇄 (기술)**

충분한 힘을 모아 방출하여 넓은 지역에 큰 피해를 입힙니다.

**회복**

진짜 모습으로 돌아가서 근처의 적들이 땅에서 발생한 충격파에 의해 부상당하게 합니다.

## 획득 방법

혈혼 절벽 레벨에서 돌 인간을 물리치고 획득합니다.

## 자세한 소개

사용 시, 돌 정령으로 변신하여 불, 냉기, 번개, 독에 대한 면역력을 얻습니다.

단단하고 무거워, 공격보다는 방어에 중점을 둡니다. 변신 중에 공격을 받으면 힘을 계속 축적할 수 있습니다.
