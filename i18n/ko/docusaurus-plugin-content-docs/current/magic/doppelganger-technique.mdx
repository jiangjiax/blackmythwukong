# 검은 신화: 오공에서의 많은 중 하나의 뽑기

검은 신화: 오공에서 "많은 중 하나의 뽑기"는 자신의 몸에서 나온 털을 이용하여 여러 개의 클론을 소환하여 전투를 돕는 액티브 주문입니다.

    src="https://image.informedainews.com/fss.gif"

/>

- **Type:** 털
- **Name:** 많은 중 하나의 뽑기
- **Cooldown:** 60초
- **Mana Cost:** 130

## 자세한 소개

이 주문은 선택받은 자가 자신의 몸에서 나온 털을 사용하여 시전합니다. 한 줌의 털을 뽑아 입에서 부수고 공중에 뱉으면, 수많은 털 원숭이가 소환됩니다.

이 털 원숭이들은 날렵하고 민첩하며, 선택받은 자와 같이 싸우고 도망칠 수 있습니다. 주인이 위협을 받을 때 상대와 교전하여 숨을 돌리거나 전략적인 후퇴를 할 수 있는 기회를 제공합니다. 하지만 이 털 원숭이들의 생명은 연약하여, 강력한 적과 마주치면 빠르게 전멸할 위험이 있습니다.
