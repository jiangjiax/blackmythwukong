# 통계

'검은 신화: 오공'의 통계에는 기본 속성과 사대 재앙 저항력이 포함되며, 다음과 같습니다:

## 기본 속성

- 체력 (Health)
- 마나 (Mana)
- 기 파워 (Qi Power)
- 공격력 (Attack)
- 방어력 (Defense)
- 기 회복 (Qi Recovery)
- 치명타 확률 (Critical Hit Rate)
- 치명타 데미지 (Critical Damage)
- 데미지 증가 (Damage Increase)
- 데미지 감소 (Damage Reduction)

## 사대 재앙 저항력

- 냉기 저항 (Cold Resistance)
- 화염 저항 (Fire Resistance)
- 독 저항 (Poison Resistance)
- 번개 저항 (Lightning Resistance)
