# 검은 신화: 오공의 황풍 능 지도

검은 신화: 오공에서, 황풍 능은 고전 중국 소설 서유기에서 영감을 받은 중요한 장소입니다.

## 서유기의 황풍 능

서유기의 원문에서, 황풍 능은 제20장 "황풍 능: 당승 곤경, 저팔계 산을 오르다"라는 제목에서 처음 언급됩니다. "팔백 리 황풍 능"이라고도 합니다.
