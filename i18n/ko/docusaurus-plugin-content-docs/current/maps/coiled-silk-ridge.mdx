# 검은 신화: 오공의 꼬인 실 능선 지도

## 서유기에서의 꼬인 실 능선

서유기의 원문에서 꼬인 실 능선은 72장 "실굴: 일곱 가지 감정이 본성을 혼란스럽게 하고, 먼지 씻는 샘: 저팔계가 통제를 잃다"에서 처음 언급됩니다.

현지의 땅의 신이 손오공에게 이 지역을 소개하면서 꼬인 실 능선 아래에 실굴이라는 동굴이 있으며, 이곳에는 일곱 마리의 여성 요괴들이 살고 있다고 언급합니다. 실굴의 남쪽에는 먼지 씻는 샘이라는 샘이 있는데, 이는 원래 일곱 선녀들의 목욕탕이었지만 요괴들에 의해 점령된 천연 온천입니다.
