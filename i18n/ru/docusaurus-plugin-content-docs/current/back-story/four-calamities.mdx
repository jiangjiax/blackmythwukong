# Четыре Бедствия

Четыре Бедствия относятся к четырем основным катастрофам во Вселенной: громовой удар, огонь, сильный холод и отравление. Когда эти бедствия накапливаются до определенного уровня, они наносят соответствующий ущерб. Можно употреблять лекарственные травы для облегчения состояния.

## Историческое Обоснование

Концепция Четырех Бедствий, вероятно, адаптирована из первоначальных Трех Бедствий, которые упоминаются в Главе Второй китайского романа "Путешествие на Запад". Три Бедствия — это ссылка на три катастрофы, с которыми должен столкнуться практикующий даосизм во время своей практики: громовой удар, огонь и ветер. Эти бедствия не только физические испытания, но и духовные испытания, которые проверяют решимость и чистоту практикующего. Добавление четвертого бедствия, отравления, в этом контексте может быть адаптацией для расширения спектра испытаний и опасностей, с которыми сталкиваются персонажи в истории.
