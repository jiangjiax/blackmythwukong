# Статистика

Статистика в 'Black Myth: Wukong' включает базовые атрибуты и сопротивления четырем бедствиям, которые следующие:

## Базовые атрибуты

- Здоровье
- Мана
- Ци
- Атака
- Защита
- Восстановление Ци
- Шанс критического удара
- Критический урон
- Увеличение урона
- Уменьшение урона

## Сопротивления четырем бедствиям

- Сопротивление холоду
- Сопротивление огню
- Сопротивление яду
- Сопротивление молнии
