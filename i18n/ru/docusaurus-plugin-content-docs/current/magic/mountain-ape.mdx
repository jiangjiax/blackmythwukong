## Ходы

**Удар Когтями (Легкая Атака)**

Легкая атака, состоящая из четырех этапов: правой когтями, левой когтями с рывком вперед, правой когтями и левой когтями, ударяющими по земле.

**Удар Молотом (Тяжелая Атака)**

Тяжелая атака, состоящая из четырех этапов: четыре раза ударяет по земле двумя кулаками, создавая сильный удар, который может оттолкнуть ближайших врагов.
