# Gewundener Seidenfadengrat Karte in Black Myth: Wukong

## Gewundener Seidenfadengrat in Reise nach Westen

Im ursprünglichen Text von Reise nach Westen wird der Gewundener Seidenfadengrat erstmals in Kapitel 72 erwähnt, das "Seidenhöhle: Sieben Emotionen verwirren die ursprüngliche Natur; Waschschmutzquelle: Zhu Bajie verliert die Kontrolle" heißt.

Der örtliche Landgott stellt Sun Wukong das Gebiet vor und erwähnt, dass sich unter dem Gewundener Seidenfadengrat eine Höhle namens Seidenhöhle befindet, in der sieben weibliche Dämonen leben. Südlich der Seidenhöhle gibt es eine Quelle namens Waschschmutzquelle, die eine natürliche Heißwasserquelle ist, die ursprünglich der Badeplatz der Sieben Schönheiten war, aber mittlerweile von den Dämonen besetzt wurde.
