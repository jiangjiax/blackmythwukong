# 黑神話悟空中的金池長老

金池長老是觀音禪院的住持。當玄奘一行路過時，他因見到玄奘的錦襴袈裟而心生貪念。最終，他在禪院被焚毀時死去，化為妖怪王Boss。他的身份和經歷反映了貪婪與罪惡的主題，暗示他生前的錯誤選擇導致了他現在的妖怪王Boss地位。

## 攻略

金池長老的戰鬥風格以高傷害和範圍攻擊為特點。在戰鬥中，玩家可以通過擊敗場地中的小僧來恢復生命值，從而降低難度。總體來說，金池長老的戰鬥機制與幽靈相似，但由於其高傷害輸出和召喚小怪的能力，玩家需要更加注意戰鬥策略。

## 招式

### 雙掌連擊

金池長老站立不動，雙掌快速連擊，造成大量傷害。建議保持距離，避免被擊中。

### 躍擊、震地、升雲掌

當金池長老與玩家有一定距離時，他會躍起並猛擊地面，造成衝擊波後接連招。玩家必須躲避最初的猛擊，以避免受到重大傷害。

<img src="https://image.informedainews.com/j2.gif" alt="Black Myth: Wukong"/>

### 衝刺、猛擊、螺旋輸出

一種緩慢的飛踢攻擊，容易預測和躲避。

<img src="https://image.informedainews.com/j3.gif" alt="Black Myth: Wukong"/>

### 召喚爆破小僧

金池長老召喚小僧追趕玩家並爆炸，造成傷害。玩家應優先消滅這些小僧，以防止金池長老恢復生命值。
