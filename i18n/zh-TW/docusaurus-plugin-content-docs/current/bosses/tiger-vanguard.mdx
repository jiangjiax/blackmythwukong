# 黑神話悟空中的虎先鋒

數百年前，前代虎先鋒在捉拿玄奘時喪命，留下兩個兒子相依為命。長兄虎兄帶領殘存的小妖們在荒廢的山中掙扎求生。幸好父親傳授了他們一些真本事，讓他們得以堅持下來。

隨著沙塵暴的平息，母石與石精靈們重建了山巒往日的輝煌。虎兄效仿父親，管理山中的妖怪群體，確保每個妖怪都有其職責，不隨意傷害生靈。後來，一些凡人在山中建立了村莊，虎兄遵循「吞食惡意闖入者，引導善良旅人」的習俗，被尊稱為「虎神」。

村民們建造了臥虎寺來供奉「虎神」，這使得虎兄的弟弟虎弟越來越不滿，認為長兄獨佔了所有的名聲，僅僅因為他早出生幾年。

又過了一百年，齊天大聖猴突然帶著一群鼠妖回到了黃風嶺。虎兄不悅，認為正是這個黃毛鼠精導致了他們父親的死亡。他慫恿虎弟驅逐鼠妖，虎弟覺得有理，便同意了。

然而，在戰鬥中，齊天大聖猴揮舞著形似佛頭的武器，召喚風刃，一擊重創了虎弟。為了救兄長的性命，虎弟不得不放下武器，跪地投降。齊天大聖猴不記仇，得知兄弟倆的過往後，任命虎弟為其麾下的先鋒。

虎弟傷癒後，得知兄長成為了先鋒，拒絕服從。兩虎大吵一架，虎弟憤怒離去，決心不再與兄長有任何瓜葛。

對於虎先鋒來說，那黃毛黃鼠狼精讓他兩次失去了家人，實在可恨。但除了每天多吃些老鼠來洩憤，他還能做些什麼呢？

## 招式

### 衝鋒猛擊

虎先鋒先以肩撞衝刺一段距離，然後躍起猛擊地面。此招相對容易完美閃避，猛擊後有露出關節的動畫。

### 三連下勾拳接猛擊

虎先鋒擺出架勢，輕輕將右臂後拉，然後身體前傾，施展旋轉踢腿以縮短距離，緊接著連續兩次前衝拳擊（右、左）。經過短暫的蓄力後，躍起猛擊地面。

虎先鋒的旋轉踢腿移動幅度較大，能迅速壓迫玩家。注意閃避的時機和方向。完美閃避兩次拳擊組合後，等待猛擊的恢復窗口進行反擊。
