# 黑神話悟空中的刃狼教官

刃狼教官在遊戲《黑神話：悟空》中作為黑風山的領袖出現，其真名為「廣智」。

廣智出現在《西遊記》原著的第十六回「觀音院僧謀寶貝，黑風山怪竊袈裟」中。他是觀音院金池長老的弟子。當唐僧師徒來到觀音院時，金池長老熱情款待了他們。悟空向長老炫耀自己的袈裟，長老心生貪念，借去房中欣賞。長老想要佔有袈裟，聽從廣智的建議，用刀殺死唐僧，但未能成功。隨後，他又聽從另一弟子廣謀的建議，命令僧人放火，意圖將唐僧師徒燒死。悟空前往南天門向廣目天王借來避火罩保護師父，並吹起一陣神風，將觀音院燒成一片廢墟。長老羞愧難當，無顏見人，撞牆而死。

## 攻略

壓制它，使其無法使用技能。

## 招式

### 衝刺雙斬

刃狼教官站立不動，蓄力片刻後揮動雙刃，隨即向前衝刺至玩家附近揮動，接著進行兩次連續的下劈攻擊，每次攻擊都伴隨著短暫的追蹤移動。此招式的第一次衝刺有較長的前搖階段，容易躲避，但第二次攻擊疑似有快慢刀，躲避和反擊時需謹慎。三次攻擊的第二次旋轉涉及手持刀刃的旋轉，形成一個圍繞身體的360度攻擊判定。
