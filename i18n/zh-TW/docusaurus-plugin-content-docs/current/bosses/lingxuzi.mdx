# 黑神話悟空中的靈虛子

靈虛子是位於黑風山觀音寺的妖王級Boss，他是一隻巨大的灰狼精，也是黑風王的同伴。

該角色的設計和能力靈感來自中國傳統神話，其中靈虛子與狼精「靈虛子」有關，以其強大的存在感和與黑風王的聯繫而聞名。

## 攻略

靈虛子的攻擊強大而直接，最多可達四連擊（基於2020年的遊戲演示）。他的攻擊有較長的啟動和恢復時間，給玩家充足的反應時間。他對火屬性極為脆弱，因此刀狼教頭「廣智」是一個極佳的變身選擇，每次攻擊都能中斷並傷害Boss。

作為一隻灰狼精，靈虛子擁有厚厚的毛皮，使他容易受到火的傷害。

## 招式

### 撕裂爪

這是靈虛子的開場招式，他會向前跑兩步，然後跳起，迅速張開雙臂進行攻擊。

<img src="https://image.informedainews.com/siliezhua.gif" alt="Black Myth: Wukong"/>

### 跳躍橫掃

靈虛子高跳後，用右前肢橫掃，落地後再用左前肢進行廣範圍橫掃。

<img src="https://image.informedainews.com/fpjhs.gif" alt="Black Myth: Wukong"/>

### 狼嚎

靈虛子後退幾步並嚎叫，造成大範圍傷害。

<img src="https://image.informedainews.com/lh.gif" alt="Black Myth: Wukong"/>

### 咬擊

當玩家靠近靈虛子時，他會向前撲咬。這一攻擊由單一動作組成，可以通過滾動輕易躲避。
