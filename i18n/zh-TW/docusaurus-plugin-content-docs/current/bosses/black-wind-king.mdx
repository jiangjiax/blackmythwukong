# 黑神話悟空中的黑風大王

黑風大王是位於黑風山黑風洞中的強大妖王。他自稱是黑風山的統治者，展現出強大的力量和兇猛、靈巧的戰鬥風格。
曾經是一個普通的妖怪，黑風大王因貪婪和對權力的渴望，最終變成了黑風山的妖王。他的故事反映了慾望和墮落的主题，象徵著在追求權力的過程中迷失自我。

## 策略

黑風大王的戰鬥風格以高爆發傷害和靈活攻擊為特點。玩家在戰鬥中必須保持警惕，利用環境進行閃避和反擊。觀察他的攻擊模式將有助於玩家找到反擊的機會，同時注意他召喚的火焰和風暴以避免受傷。

## 招式

### 重擊攻擊

黑風大王以強大的力量猛擊地面，造成範圍傷害。玩家應迅速閃避以避免被擊中。

### 火焰拳

他發出一記強力的左拳，然後用右爪向前突進，試圖抓住並吞噬敵人。這一招有較長的恢復時間，玩家可以趁機進行傷害。

<img src="https://image.informedainews.com/bwk2.gif" alt="Black Myth: Wukong"/>

### 地面火焰噴發

黑風大王擊打地面，引發地下的火焰噴發。玩家必須迅速躲避以避免被燒傷。

<img src="https://image.informedainews.com/bwk3.gif" alt="Black Myth: Wukong"/>

### 黑風衝刺

他化作一股黑暗的風，快速移動並衝向玩家，造成大量傷害。玩家應注意風的移動並相應地閃避。
