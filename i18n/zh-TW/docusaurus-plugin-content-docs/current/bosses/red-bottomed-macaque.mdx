# 赤尻馬猴在黑神話悟空

在黑神話悟空中，赤尻馬猴是小雷音寺地區的BOSS。

在《西遊記》中，赤尻馬猴並不是特指一隻猴子，而是一個種類。在花果山的四健將中，有兩隻是赤尻馬猴，而在混世四猴中，也有一隻是赤尻馬猴。原著中描述其「曉陰陽，會人事，善出入，避死延生」。

## 攻略

壓制它，使其無法使用技能。

## 招式

### 第一階段

#### 起手招式

赤尻馬猴善於埋伏；過場動畫結束後會立刻攻擊玩家，注意閃避。

<img src="/img/bosses/yjdqss.gif" alt="Black Myth: Wukong"/>

#### 避死延生

讀取玩家指令，可以躲避重擊。

<img src="/img/bosses/yjdbsys.gif" alt="Black Myth: Wukong"/>

#### 強化劍氣

赤尻馬猴實際上只有兩個階段，但每個階段都有一個爆氣。爆氣後，劍會發出白光，其技能、攻擊慾望、攻擊速度和技能速度都會大大提高。赤尻馬猴的第一階段爆氣是在一分鐘內沒有被打到半血時觸發。

<img src="/img/bosses/yjdqhdq.gif" alt="Black Myth: Wukong"/>

#### 半血求饒

假裝求饒，實則埋伏。

<img src="/img/bosses/gdqr.gif" alt="Black Myth: Wukong"/>

### 第二階段標記

#### 逃脫

在打到半血後一分鐘內觸發：赤尻馬猴化作一團霧氣逃脫，進入第二階段。

<img src="/img/bosses/ejdtp.gif" alt="Black Myth: Wukong"/>

#### 起手招式

赤尻馬猴起飛，在空中釋放黑色羽毛後衝向玩家。

<img src="/img/bosses/ejdqsf.gif" alt="Black Myth: Wukong"/>

#### 第二階段強化劍氣

在打到三分之一血後一分鐘內觸發。在第二階段爆氣形態下，攻擊附帶瞬移，速度超快，攻擊更加猛烈，適合高手挑戰。
