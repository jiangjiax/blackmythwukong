# 山猿於黑神話悟空

- **類型：** 變身
- **名稱：** 山猿
- **冷卻時間：** --
- **法力消耗：** --

## 招式

**爪擊（輕攻擊）**

一種輕攻擊招式，包含四個階段：右爪、左爪帶前衝、右爪、左爪砸地。

**錘擊（重攻擊）**

一種重攻擊招式，包含四個階段：雙拳四次砸地，產生強烈衝擊，可將附近敵人推開。

## 獲取方式

## 詳細介紹

使用後，命中注定之人將變身為巨猿，身材高大，移動速度快。變身將推開所有附近敵人。變身持續時間：30秒。
