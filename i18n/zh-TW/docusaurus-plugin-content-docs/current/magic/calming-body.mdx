# 黑神話悟空中的定身法

在黑神話悟空中，「定身法」是一門非凡技藝，屬於主動咒術，通過繪製一個神秘陣法來驅趕弱小生物並增強飲酒效果。靈感源自原著《西遊記》第五十回中描述的「鎮靜身法」。

- **類型：** 非凡技藝
- **名稱：** 定身法
- **冷卻時間：** 90秒
- **法力消耗：** 60

## 詳細介紹

有緣人在一個極寒冰封之地偶然習得此項保命的非凡技藝。

只需持杖在地上畫一個圈，便可驅寒歇腳，備戰敵人。

站在圈內能持續增加杖勢，並加速精力恢復；若此時飲用溫酒，效果更佳。

火圈持續時間較長，即便短暫出圈戰鬥，亦可隨時返回圈內尋求再戰之機。

一旦時限到達，陣法自然消散。
