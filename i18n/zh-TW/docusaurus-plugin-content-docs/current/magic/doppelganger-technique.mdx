# 分身術在黑神話悟空

在黑神話悟空中，「分身術」是一種主動法術，能夠召喚多個分身協助戰鬥，利用自身毛髮來實現。

- **類型：** 毛髮
- **名稱：** 分身術
- **冷卻時間：** 60秒
- **法力消耗：** 130

## 詳細介紹

此法術由被選中者利用自身毛髮施展。通過拔取一把毛髮，咀嚼成碎片，然後吐向空中，便能召喚出眾多毛猴分身。

這些毛猴分身靈活敏捷，能夠戰鬥和逃逸，與被選中者本人相似。當主人受到威脅時，它們會與對手交戰，提供喘息或戰略撤退的機會。然而，這些毛猴分身的生命脆弱；面對強大且強力的敵人，它們可能迅速被消滅。
