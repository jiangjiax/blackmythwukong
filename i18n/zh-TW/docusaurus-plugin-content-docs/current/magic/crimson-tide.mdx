# 赤潮 於 黑神話悟空

- **生命值：** 576
- **類型：** 變身
- **名稱：** 赤潮
- **冷卻時間：** 無
- **法力消耗：** 無

## 招式

**月斬（輕攻擊）**

輕攻擊招式，刀刃上的火焰如滿月般舞動，連續斬擊對手。

**逐月（重攻擊）**

重攻擊招式，輕如火焰，追逐月亮的流星。

閃避不同方向後，接續重攻擊可演變出新的變化。

**月碎（被動能力）**

閃避後，攻擊力在短時間內顯著提升。

## 獲取方式

擊敗黑風山關卡中的BOSS廣智（刀狼教頭）獲得。

## 詳細介紹

使用後變身為狼妖，獲得火焰免疫。

火浪洶湧逼人。攻擊可附加火疫。

招式附加火疫，可持續燃燒對手。

變身期間，進入火焰免疫狀態。
