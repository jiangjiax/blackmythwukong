# 安身術於黑神話悟空

在黑神話悟空中，安身術是一門非凡的技藝，一種主動咒語，能夠將目標怪物鎖定在原地，使其無法動彈。

- **類型：** 奧術技藝
- **名稱：** 安身術
- **冷卻時間：** 30秒
- **法力消耗：** 60

## 詳細介紹

在命中注定者的掌心，鐫刻著一個金色的「定」字，賦予他們操縱安身術的非凡能力。

只需輕輕一指，低語咒語，便能使對手動彈不得，無法移動。

然而，擁有強大神力的強大敵人能夠更快地掙脫這種束縛，需要謹慎對待。

如果戰鬥激烈，對手的攻擊連綿不絕，則難以長時間使其定身，效果甚微。

或許施展此技的最佳時機，是當對手靜止不動、倒地或被擊退至虛弱狀態時。
