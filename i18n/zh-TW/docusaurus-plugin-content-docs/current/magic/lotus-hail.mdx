# 蓮花冰雹於黑神話悟空

- **類型：** 變身
- **名稱：** 蓮花冰雹
- **冷卻時間：** --
- **法力消耗：** --

## 招式

**殘殺（輕攻擊）**

輕攻擊招式，揮動手中的刀刃，切割對手。

**自殘（重攻擊）**

重攻擊招式，傷害自身，將體內的卵附著在刀刃上。充能後釋放，最多可疊加四層。

每次後續的輕攻擊命中對手會消耗一層卵。卵附著在對手的傷口上，孵化成成蟲潛伏，並在一定時間後死亡。

**勝利（法術）**

當對手已經被毒蟲寄生時，命令它們集體爆炸。

可以瞬間對對手造成嚴重創傷，同時部分蟲群返回治療一定量的生命。

**撤退（恢復技）**

完全自爆，造成範圍傷害。

在恢復真身時，也可以趁機進行勝利，引爆對手身上未過期的毒蟲。

## 獲取方法

在紫雲山落花莊擊殺相關的boss米道士後獲得。

使用後，變身為蟲妖，獲得對毒的免疫。
