# 如意金箍棒

如意金箍棒是明代神話小說《西遊記》中孫悟空使用的武器，《西遊記》是一部中國古典神話幻想小說，講述了神與魔的故事。

## 介紹

小說中孫悟空的武器正式名稱為「如意金箍棒」，別稱有金箍如意棒、金箍鐵棒、如意棒、金箍棒、靈陽棒、鐵棒，後者是使用頻率最高的簡稱。據《西遊記》記載，如意金箍棒「兩頭是兩個金箍，中間乃一段烏鐵；緊挨箍有鐫成的一行字，喚做『如意金箍棒』，重一萬三千五百斤」。東海龍宮龍婆、龍女道：「這叫做個『神珍鐵』」。王勃《廣州寶莊嚴寺舍利塔碑》：「誠心所至，冥契何速，神珍即現」。孫悟空得到神珍後，此物便隨心意變化，可大可小，可長可短，真正做到了「如意」二字。

## 起源

「金環杖說」。在《大唐三藏取經詩話》中，猴行者沒有武器，而是依靠大梵天國王所賜的金環杖捉拿妖怪。

「金剛杵與金環杖混合說」。日本學者認為，密宗的金剛杵形象影響了《西遊記》中孫悟空的形象，從而創造出了金箍棒。

「金棍與鐵棍說」。在元代雜劇《西遊記》中，孫悟空使用的是一根可以變化形態的金棍；在《西遊記平話》中，孫悟空使用的是鐵棍，這對《西遊記》產生了直接影響。

「精骨棒說」。周汝昌先生認為，「金箍棒」其實是「精骨棒」，由於讀音相近而被誤解，從而與「金箍」聯繫在一起。

「如意珠說」。「如意珠」是一種佛教寶物，又稱「摩尼珠」。據說擁有此珠可以遍觀十方世界，可以隨心變化，可以消除一切災難和苦惱。《大智度論》說此珠生出於龍王腦中。《佛說大乘無量壽莊嚴經》說龍王以如意珠置於左耳之中。
