# 龍

龍是一種傳說中的生物，在中國官方、宗教和民間信仰中被共同崇拜，在文學創作中享有極高的人氣。

## 《黑神話悟空》中的龍

根據遊戲的宣傳視頻，我們瞭解到了亢金龍和涇河龍王的兩個兒子癡鱗龍和嘯虯龍。

## 《西遊記》中的龍

在《西遊記》中，最早出現的龍是四海龍王，孫悟空為了獲得武器和盔甲而拜訪他們。在獲得金箍棒後，他又進一步要求一套好盔甲，這導致了其他三海龍王的到來。接下來是涇河龍王，他在夢中向唐太宗求助。

隨後，敘述介紹了白龍，即玉龍三太子，在鷹愁澗被觀音救下，以及托龍，他抓住了唐三藏，導致涇河龍王九子的介紹。

在小說中，龍經常作為掌管雨水和水體的神靈出現，每條河流和井都有自己的龍王。此外，它們還被描繪為玉帝的坐騎，或作為其他神靈的守護者和使者。

## 現實中的龍

考古發現表明，由於中國原始的自然崇拜，魚龍、蛇龍、豬龍和鱷龍等龍圖騰早在7000年前就已經出現。現代龍的形象特徵是鹿角、駝頭、兔眼、蛇頸、蜃腹、魚鱗、鷹爪、虎掌、牛耳，這一獨特的形象與神靈的使者或神聖坐騎等角色有關。

需要注意的是，龍控制風雨的能力與佛教的中國化密切相關。然而，龍作為雷神的崇拜在6000年前的文獻中就有記載，包括《山海經》、《五帝本紀》和《說文解字》。

在佛教經典中，如晉代翻譯的《妙法蓮華經》中，有八位龍王，包括掌管降雨的海龍王、娑伽羅龍王。然而，需要注意的是，佛教文本中的龍通常指的是那伽，這些蛇形生物在印度神話中早於佛教存在。這些生物被納入佛教，後來被翻譯成中文為龍或蛇。隨著佛教在中國的中國化，這些神靈在寺廟中的形象開始與其印度起源不同，並在流行文化中融入獨特的中國形式。
