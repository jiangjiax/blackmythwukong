# 黑神話悟空中的冥想地點

遊戲中設有一系列冥想地點，玩家首次在此冥想可獲得一顆火花。以下是所有冥想地點的位置：

## 黑風山

- 狼林之森的涼亭 - 俯瞰狼林之森的破舊木結構內部

- 竹林後山的洞穴 - 竹林後山的洞穴中

- 黑風洞的懸崖 - 黑風洞中俯瞰深淵的木製平台

## 黃風嶺

- 沙門村的祭壇 - 沙門村佛壇前

- 碎石露台的山谷 - 碎石露台前的山谷

- 臥虎寺的雕像 - 臥虎寺內的雕像下

- 枕石露台的懸崖 - 枕石露台上的枯松樹下

- 黃風陣的洞穴 - 黃風陣中的佛洞前

- 沙門邊界的岩石 - 沙門邊界的岩石上

## 新西域

- 鏡海的樹蔭 - 俯瞰鏡海的大樹下

- 寶塔界的底部 - 擊敗智言隊長後的寶塔界底部

- 戒律廊的雕像 - 戒律廊的佛像前

- 正念崖的小徑 - 通往正念崖金色壁畫的小徑上

- 新雷音寺的大殿 - 新雷音寺的大殿內

## 盤絲嶺

- 碎玉池的懸崖 - 俯瞰碎玉池的懸崖上

- 中層的樹蔭 - 中層的樹蔭下

- 底層的深處 - 底層洞穴的深處

- 虎蛇蟲林的懸崖 - 虎蛇蟲林的懸崖上

- 黃花亭的松樹 - 黃花亭的松樹和傘樹之間

- 紫雲山的岩礁 - 紫雲山的岩礁上

## 火焰山

- 四季營的浮雕 - 四季營的浮雕上

- 翡翠殿的佛像 - 翡翠殿的坐佛前

- 灰燼通道三的岩礁 - 灰燼通道三的岩礁上

- 煉化坑的洞穴 - 煉化坑的石屏風前

找到所有24個冥想地點可解鎖風景探索者成就/獎杯。首次在每個地點冥想可獲得一顆火花，用於解鎖技能。
