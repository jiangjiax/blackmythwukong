# 黑神話悟空中的盤絲嶺地圖

## 西遊記中的盤絲嶺

在《西遊記》原著中，盤絲嶺首次出現於第七十二回，題為「絲洞七情迷本性，濯垢泉八戒忘形」。

當地土地神向孫悟空介紹該地區，提到盤絲嶺下有一個名為絲洞的洞穴，居住著七名女性妖怪。絲洞南面有一處名為濯垢泉的溫泉，原本是七仙女的沐浴池，後被妖怪佔據。
