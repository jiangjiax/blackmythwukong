## 黑神話悟空地圖索引

歡迎來到**黑神話悟空**的地圖索引頁面！在這裡，您將找到關於遊戲中各種地圖的詳細資訊、敵人介紹、隱藏區域和策略提示。黑神話悟空共有八張地圖，分別是：

- [黑風山](/docs/blackwindmountain)
- 花果山
- [黃風嶺](/docs/huangfeng-ridge)
- [小西天](/docs/little-western-heaven)
- 紫雲山
- [盤絲嶺](/docs/coiled-silk-ridge)
- 火焰山

探索這些多樣化的地點，揭開它們的秘密，並準備好迎接與強大敵人的激戰，在**黑神話悟空**的神祕世界中展開您的冒險！

## [冥想點](/docs/meditationspots) 在黑神話悟空中

除了多樣化的地圖外，**黑神話悟空**還設有多個[冥想點](/docs/meditationspots)，玩家可以在這裡休息並恢復體力。這些[冥想點](/docs/meditationspots)通常位於隱秘的角落或寺廟中，為玩家提供喘息的機會。在這裡，您可以恢復生命值、整理裝備，並為下一個挑戰做好準備。
