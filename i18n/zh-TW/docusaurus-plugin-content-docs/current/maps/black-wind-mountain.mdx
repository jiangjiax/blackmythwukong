# 黑神話悟空中的黑風山地圖

歡迎來到**黑神話悟空**中的黑風山地圖頁面！這片險惡之地居住著強大的敵人、豐富的背景故事以及各種收集品，邀請玩家深入探索並揭開其秘密。

## 黑風山區域

黑風山分為幾個獨特的區域，每個區域都有其獨特的挑戰和敵人：

- 狼林
- 竹林
- 古觀音寺
- 黑風洞

## 黑風山中的Boss

在黑風山中，玩家將遭遇各種Boss，每個都提供獨特的挑戰：

- [刀狼教頭](/docs/blackwindking)
- [凌虛子](/docs/lingxuzi)
- 白衣秀士
- [黑風王](/docs/blackwindking)
- [金池長老](/docs/elderjinchi)

## 黑風山中的收集品

玩家可以在黑風山中收集各種物品，包括：

- 靈芝
- 猴頭菇
- 烏頭
- 蓮花
- 猴頭菇
- 茯苓
